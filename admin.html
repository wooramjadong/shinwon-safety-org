Google Sheets 연동 설정 방법
  1. Google Drive 폴더에서 새 Google Sheets 생성
  2. 시트에서 도구 → Apps Script 클릭
  3. 아래 Apps Script 코드를 복사해 붙여넣기
  4. 배포 → 새 배포 → 웹 앱 선택
  5. 실행: 나 / 액세스: 모든 사용자 설정 후 배포
  6. 생성된 웹 앱 URL을 위의 Apps Script URL 입력란에 붙여넣기
  7. 설정 저장 클릭 → 각 현장 URL 배포
